# AI Impact Analysis Using SQL

## Project Overview

This project analyzes the impact of AI tools on student productivity, academic performance, and career confidence using SQL.

## Tools Used

- SQL Server Management Studio (SSMS)
- SQL

## Business Questions

1. Which age groups save the most time using AI?
2. Which majors benefit the most from AI?
3. Does AI usage increase career confidence?
4. How does AI affect academic performance?

## Skills Demonstrated

- Data Cleaning
- Aggregations
- Window Functions
- Ranking Functions
- Business Analysis

## Files Included

- AI_Impact_Analysis.sql
- AI_Impact.csv
- Project Screenshots

## Key Insights

- Students reporting higher AI usage generally showed stronger career confidence.
- Productivity gains varied across majors.
- AI adoption was associated with improved academic outcomes.
